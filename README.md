# Calculadora em JavaScript
Projeto de uma calculadora simples desenvolvido para fins de estudo em programação web.

## Objetivo
Praticar lógica de programação e manipulação do DOM utilizando JavaScript, além de estruturar e estilizar páginas com HTML e CSS.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar valores

## Observações
Este projeto foi criado apenas para fins de aprendizado e prática.
